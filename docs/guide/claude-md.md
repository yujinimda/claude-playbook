# CLAUDE.md 작성법

> 프로젝트의 CLAUDE.md를 어떻게 구성하면 좋은지 가이드합니다.

## 기본 구조

```markdown
# 프로젝트명

## 역할
Claude가 이 프로젝트에서 해야 할 역할 정의

## 스택
- 언어/프레임워크
- 주요 라이브러리

## 규칙
팀이 합의한 코딩 규칙

## 스킬
사용 중인 스킬 목록
```

## 작성 원칙

- **500줄 이내** 유지 (Progressive Disclosure 원칙)
- 자주 바뀌는 내용은 별도 파일로 분리
- 중요도 순으로 배치 (Claude는 앞부분을 더 주의 깊게 읽음)
