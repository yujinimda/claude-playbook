# 룰 관리법

> 팀 코딩 룰을 어떻게 정의하고 관리하는지 설명합니다.

## 룰이란?

Claude Code가 코드를 작성할 때 따라야 할 팀 규칙입니다. CLAUDE.md에 직접 넣거나 별도 파일로 관리합니다.

## 파일 형식

```markdown
# [룰 이름]

## 목적
이 룰이 해결하는 문제

## 규칙
- 구체적인 규칙 항목들

## 예시
좋은 예 / 나쁜 예
```

## curated vs custom

- `curated/` — 외부(커뮤니티, 마켓)에서 가져온 검증된 룰. 출처 링크 필수.
- `custom/` — 팀이 직접 만든 룰. 해결하는 문제 설명 필수.
